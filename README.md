### Contact Management System

A simple Contact Management System built using Spring Boot, MySQL, HTML, and JavaScript.
This project demonstrates full-stack development with REST API integration and frontend-backend communication.

### Features
Submit contact form (Name, Email, Message)
Store data in MySQL database
REST API to fetch all contacts
Display contacts in a dynamic HTML table
Uses JavaScript Fetch API for frontend-backend communication
🛠️ Tech Stack

### Backend:
Java
Spring Boot
Spring Data JPA
MySQL

### Frontend:
HTML
JavaScript (Fetch API)
