### Contact Management System ###

A simple Contact Management System built using Java, Spring Boot, MySQL, HTML, Spring Security, and JavaScript.

This project demonstrates full-stack development with REST API integration, frontend-backend communication, database management, and secure authentication using Spring Security.

### Features
 ## Contact Management
Submit contact form (Name, Email, Message)
Store contact details in MySQL database
REST API to fetch all contacts
Display contacts in a dynamic HTML table
JavaScript Fetch API for frontend-backend communication

## Security Features (Task 4)

Spring Security Authentication
Spring Security Authorization
Role-Based Access Control (ADMIN)
Secure Login and Logout
BCrypt Password Encryption
Protected REST API Endpoints
Session-Based Authentication
